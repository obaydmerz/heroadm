# How to use LittleAdm Tools

### List of Tools
    - Contacts
        * Auth
            - Model: This Model is Auth Model
        * Aborter: Execute Your Function And Abort if it Return false
        * CRUDPermi: Select Peoples Have Role To See
    - CrudDefiner
        * Column: Generate Column Definition
        * Pusher: Generate Push Array
        * Validator: Generate Validate Array
    - Locale
        * LocaleTrt: Translation
    - Schema
        * LADSchemaManager: Manage Database
        * Table: Generate Table Class
    - ArrayCompb: Manage & Integrate Arrays
    - Crud: Crud Definer
    - CrudRoute: Route For Cruds
    - FileDefiner: File Manager
    - ObjectComp: Manage & Integrate Objects

### Warning
    Lost Things In List used By System LAD
    Please Don't Use

### Namespace of Tools
The Namespace of Tools is App\LittleADM\Tools
